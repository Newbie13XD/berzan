Webbläsarkompatebilitet med Berzan.js

Webbläsare som officiellt stöds av Berzan.js

  • Mozilla Firefox
  • Google Chrome/Chromium
  • Microsoft Edge
  • Epiphany/GNOME Browser
  • Qutebrowser
  • SeaMonkey

Buggar som uppstår när Berzan.js används i dessa webbläsare ska rapporteras på GitHub.

Webbläsare som inte stöds av Berzan.js

Dessa webbläsare stödjer inte modern JavaScript, och kommer därför inte förbi laddningskärmen.

Jag var inne på att försöka få Berzan.js att fungera i Internet Explorer 11 ett tag, men även när jag lyckades få Internet Explorer 11 att acceptera min JavaScript kod så visade det sig att problemen inte stannade där, så detta lär inte hända. Koden som används för att åstadkomma detta finns att tillgå på GitHub under grenen "ie11-experiment".

  • Microsoft Internet Explorer
  • Wine Internet Explorer
